# Actividad Formativa 2: Simulación y análisis de señales

## Descripción
Este proyecto implementa la Transformada Rápida de Fourier (FFT) para analizar una señal senoidal, permitiendo observar cómo una señal en el dominio del tiempo se descompone en el dominio de la frecuencia.

## Contenido del repositorio
- `analisis_fiurier.py`: Código principal que realiza el cálculo de la FFT.
- `Figura_1.png`: Visualización de la magnitud y fase del espectro.

## Conclusión
La Transformada de Fourier actúa como un prisma matemático; al procesar nuestra señal senoidal de 5 Hz, logramos identificar un pico claro en el espectro de frecuencias, confirmando que la FFT es capaz de extraer información esencial de una señal compleja.# Analisis-de-se-ales
